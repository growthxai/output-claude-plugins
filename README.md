# Output.ai Claude Plugins
The repository contains Claude Plugins for Output.ai.

Learn more about Output.ai at [output.ai](https://docs.output.ai).

## Installation
```bash
claude
>Â /plugin marketplace add growthxai/output-claude-plugins
> /plugin install outputai@outputai
> /plugin install outputai-flow-migrator@outputai
```

Afterwards you should see slash commands in the Claude CLI autocomplete.

```bash
/outputai:plan_workflow
/outputai:build_workflow
/outputai:debug_workflow
# ... and more
```

as well as skills and agents.
